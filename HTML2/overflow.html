Absolute and Percentage Layout

The page simply takes up a fixed percentage of the browser screen. The proportions are all maintained, no matter what the screen size is. Having the page resize with the browser works, but it’s not a complete solution. When the browser window is small enough, the text will no longer fit without some ugly bleed-over effects. You can fix this with the overflow attribute, but then you will have scrollbars in your smaller elements. The key to any absolute positioning (even this flexible kind) is math.
